# My MERN Project

A full-stack web application built with the MERN stack.

## 📖 About
This is a MERN stack application with a React frontend and Node.js + Express backend connected to MongoDB.

## 🚀 Tech Stack
- **Frontend:** React, Vite
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose

## ⚙️ Installation & Setup

1. **Clone the repo**
```bash
git clone https://github.com/Veronica00-byte/my-mern-project.git
cd my-mern-project

2. **Setup Backend**
```bash
cd server
npm install

Create a .env file inside the server folder and add:
PORT=5000
MONGO_URI=your_mongodb_connection_string

Start the backend:
npm run dev

1.Setup Frontend
cd client
npm install
npm run dev

Open in browser
Frontend: http://localhost:5173
Backend: http://localhost:5000
